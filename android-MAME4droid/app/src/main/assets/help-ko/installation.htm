첫 실행: ROM을 어디에 저장할지

MAME4droid를 처음 시작하면 ROM 파일(그리고 MAME의 나머지 데이터: artwork, 샘플, 저장 상태 등)을 어디에 보관할지 묻습니다. 두 가지 선택지가 있습니다:

  • 내부:MAME4droid가 자체 비공개 앱 저장소 안에 빈 폴더를 만들어 사용합니다. 그런 다음 PC에서 USB/ADB로, 또는 앱의 비공개 폴더를 탐색할 수 있는 파일 관리자 앱으로 ROM을 직접 복사합니다. 이 데이터는 앱을 제거하면 자동으로 삭제됩니다.
    권장하지 않음:이 폴더는 앱 전용 샌드박스 안에 있으므로 외부 런처/프론트엔드나 대부분의 파일 관리자는 접근하지 못할 가능성이 큽니다 — Android의 저장소 보안 모델은 앱의 비공개 저장소가 아니라 (아래의) SAF를 통해 부여된 폴더를 전제로 합니다.
  • 외부 저장소:Android 표준 폴더 선택기를 사용해 기기의 아무 곳(SD 카드, 이미 ROM에 사용 중인 폴더, 클라우드 동기화 폴더 등)에 있는 기존 폴더를 선택합니다. MAME4droid는 그 폴더를 읽고 쓸 권한을 Android에 요청하고 다음번을 위해 기억합니다. 이 데이터는 앱을 제거해도 남으며, PC 없이 아무 일반 파일 관리자로 파일을 자유롭게 넣고 뺄 수 있습니다.

중요:외부 저장소에서는 선택한 폴더가 바로 ROM 폴더 자체입니다 — MAME4droid는 그 안에 roms 하위 폴더를 만들지 않습니다. ZIP으로 압축한 ROM 파일을 선택한 폴더에 직접 넣으세요. 같은 폴더 안에 다음 선택적 하위 폴더를 직접 만들 수도 있습니다:

artwork/ 베젤/마퀴 artwork 세트용
samples/ 사운드 샘플용
software/ 소프트웨어 목록 미디어용(8비트 가정용 컴퓨터처럼 이동식 소프트웨어가 있는 비아케이드 시스템에서 사용하는 카세트/디스크 이미지)
sta/ 저장 상태용, 저장 상태를 ROM 경로에도 활성화한 경우에만(아래 참조)

그 외 모든 것(MAME 설정, 하이스코어, NVRAM, 스냅샷)은 ROM이 어디에 있든 항상 MAME4droid 자체의 내부 데이터에 저장됩니다.

내부 저장소를 사용하면 MAME4droid는 대신 자체 비공개 앱 저장소 안에 아래의 전체 폴더 트리 (roms/ 하위 폴더 포함)를 만듭니다.

폴더 선택기에 대하여(Storage Access Framework)

Android 10부터 앱은 더 이상 공유 저장소의 아무 곳이나 자유롭게 읽고 쓸 수 없습니다 — 사용자가 Android 자체 선택기를 통해 명시적으로 부여한 폴더에만 접근할 수 있습니다(이를 Storage Access Framework, 줄여서 SAF라고 합니다). 위에서 외부 저장소를 선택하거나 나중에 ROM 폴더를 변경할 때 열리는 것이 그 선택기입니다. 직접 파일 접근보다 조금 느리므로, MAME4droid는 폴더 목록을 캐시하여 ROM이 색인되면 일상 사용에서 빠르게 유지되도록 합니다. 방금 추가한 ROM이 보이지 않거나 폴더 관련 문제가 있다면, 옵션->설정->일반에서 ROM 폴더 색인 캐시를 끄거나 ROM 폴더 다시 스캔을 실행하세요.

팁:ROM 폴더를 평평하게 유지하고(ROM을 하위 폴더에 묻지 마세요) 관련 없는 압축되지 않은 파일을 섞지 마세요 — 폴더에는 정말로 ZIP으로 압축한 MAME ROM 세트만 있어야 합니다.

나중에 ROM 위치 변경

처음에 제대로 선택할 필요는 없습니다. 옵션->설정->일반->ROM 경로 변경으로 이동해 같은 내부/외부 저장소 선택을 다시 실행하세요(이후 앱이 재시작됩니다). 새 외부 폴더로 전환하는 경우, 먼저 ROM 파일을 실제로 그곳으로 옮기거나 복사하는 것을 잊지 마세요.

저장 상태를 외부 저장소의 ROM과 함께 두고 싶다면(나중에 제거할 때 Android가 정리하지 않도록), 옵션->설정->일반->저장 상태를 ROM 경로에를 활성화하세요.

고급: 전체 설치 재배치

ROM 폴더와 별개로, 숙련된 사용자는 옵션->설정->고급->설치 경로에서 MAME4droid의 전체 데이터 폴더(ROM뿐만 아니라 아래에 나열된 모든 설정/지원 폴더)를 특정 경로로 재배치할 수 있습니다. 이것은 선택이 아니라 직접 입력하며, 고급 사용 전용입니다: 유효하지만 의도하지 않은 경로를 입력하면 MAME4droid가 이를 완전히 새로운 빈 설치로 취급합니다. 특별히 필요하지 않으면 그대로 두세요.

폴더 구조

이것은 MAME4droid의 내부 데이터 폴더 트리입니다(내부 저장소에서는 그대로 사용됩니다. 외부 저장소에서는 위에서 설명한 대로 아래의 roms/artwork/samples/software만 실제로 선택한 폴더에 존재하며, 그곳의 폴더 이름에는 끝의 슬래시가 없습니다). 데스크톱 MAME가 사용하는 것과 동일한 구조이므로 기존 artwork/샘플 컬렉션을 재사용할 수 있습니다. 이 자산을 어디서 찾는지는 리소스 및 자산 섹션을, ROM 명명 규칙은 지원 게임을 확인하세요.

artwork/ -> Artwork 디렉터리
cfg/ -> MAME 설정 파일 디렉터리
hi/ -> 하이스코어 디렉터리
nvram/ -> NVRAM 파일 디렉터리
roms/ -> ROM 디렉터리
samples/ -> 샘플 디렉터리
snap/ -> 화면 스냅샷 디렉터리
sta/ -> 저장 상태 디렉터리
