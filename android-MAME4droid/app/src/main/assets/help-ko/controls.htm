조작

이것은 기본 설정이며 컨트롤러 매핑을 변경하면 항상 적용되지 않을 수 있습니다.

가상 패드

  • 화면상 컨트롤러 및 버튼
  • 아날로그 조작용 터치 마우스. 첫 번째 손가락으로 이동. 두 번째 손가락이 왼쪽 버튼. 세 번째 손가락이 오른쪽 버튼.
  • 라이트건(화면 탭. 옵션->설정->입력->터치 라이트건에서 활성화해야 함)

게임 패드

  • 컨트롤러의 A, B, C, D, E, F 버튼은 버튼 1~6에 대응
  • SELECT(코인) 버튼으로 코인 투입/크레딧 추가
  • START 버튼으로 1P 게임 시작

특수

  • START+위로 2P 게임 시작. START+오른쪽으로 3P 게임 시작. START+아래로 4P 게임 시작
  • SELECT+위로 2P 크레딧 투입. SELECT+오른쪽으로 3P 크레딧 투입. SELECT+아래로 4P 크레딧 투입
  • START+SELECT로 게임의 MAME 메뉴(비디오 옵션, DIP 스위치 등)에 접근
  • A+START는 빨리 감기(누르는 동안 스로틀링 비활성화 및 최대 프레임 스킵으로 게임 실행).
  • B+START는 UI 컨트롤 전환(키보드 입력이 있는 에뮬레이트 시스템에서 UI 컨트롤 활성화/비활성화).
  • R1+START로 상태 불러오기
  • L1+START로 상태 저장
  • START+A+B는 서비스 모드
  • SELECT+A+B는 소프트 리셋
  • L1 또는 A+위는 게임 페이지 업
  • R1 또는 A+아래는 게임 페이지 다운

참고: UI 탐색에는 SELECT(코인) 또는 START를 사용하세요.
