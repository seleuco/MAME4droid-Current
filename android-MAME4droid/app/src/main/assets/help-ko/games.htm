지원 게임

MAME4droid (Current)는 현재 데스크톱 MAME의 게임 ROM을 사용합니다. 참고: 이는 MAME4droid (0.139u1)에서 사용하는 ROM과 다릅니다. 이 ROM들은 서로 바꿔 사용할 수 없습니다.

ROM 위치 및 명명 규칙

ROM은 MAME4droid 설치의 "roms" 폴더에 들어갑니다. 기본값(첫 실행 시 선택한 내부 저장소)으로는 앱 자체의 비공개 폴더이며, 보통 /storage/emulated/0/Android/data/com.seleuco.mame4d2024/files/roms이고, Android 디버그 브리지(ADB) 도구를 사용해 PC에서 그곳으로 ROM을 복사합니다.

또는 첫 실행 시(또는 나중에, 아래 참조) 외부 저장소를 선택하고 Android 자체 폴더 선택기를 사용해 기기의 아무 폴더나 대신 선택할 수 있습니다. 그곳에서 읽는 것은 조금 느리지만, 앱을 제거해도 아무것도 삭제되지 않으며 아무 일반 Android 파일 관리자로 ROM을 넣고 뺄 수 있고 PC가 필요 없습니다. 두 옵션의 전체 세부 사항은 설치 섹션을 참조하세요.

또한 artwork/samples/software 하위 폴더를 직접 만들어 artwork/샘플 같은 다른 선택된 리소스를 추가할 수 있습니다(정확한 위치는 설치 참조). 숙련된 사용자라면 메인 설치 폴더의 .ini 파일을 편집해 모든 경로를 완전히 사용자 지정할 수도 있습니다.

첫 선택에 묶이지 않습니다: ROM은 언제든 옵션->설정->일반->ROM 경로 변경에서 재배치할 수 있습니다.

참고:파일 및 디렉터리 이름은 대소문자를 구분합니다. MRDO.ZIP은 mrdo.zip과 같지 않습니다. 항상 파일 이름을 소문자로 한 ZIP 압축 ROM을 사용하세요.
