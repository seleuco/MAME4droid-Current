  • 2026-07-08 버전 1.37 P2P 넷플레이 추가:
    - 두 가지 모드:록스텝(완벽히 동기화, 입력 지연이 큼)과 롤백(저지연, 빠른 반응이 필요한 게임에 이상적).
    - 자동 폴백:저장 상태 크기/RAM 예산에 맞지 않거나 기기 간 저장 상태 레이아웃 불일치를 감지한 롤백 세션은 자동으로 록스텝으로 폴백합니다.
    - 게임 중 재동기화:롤백 세션이 눈에 띄게 어긋나면 넷플레이 대화상자에서 두 기기를 완벽한 동기화로 강제로 되돌립니다.
    - 호스트만 세션 모드를 선택합니다; 클라이언트는 참가해 자동으로 채택할 뿐입니다.
    앱 내 도움말을 새로 고치고 확장: 새 넷플레이와 렌더러 및 셰이더 페이지, 그리고 Android 10부터 사용되는 Storage Access Framework (SAF) 폴더 선택기를 다루는 업데이트된 설치 가이드.
  • 2026-24-4 버전 1.35 상당한 개선이 있는 새 네이티브 비디오 렌더러 추가:
    - 벡터 게임:더 선명하고 진짜 같은 외관을 위해 렌더링 품질을 크게 향상.
    - 셰이더:사용자 지정 셰이더가 이제 게임 화면에만 적용되어 MAME UI/OSD를 깨끗하게 유지.
    - 성능:고해상도 artwork와 베젤을 사용할 때 훨씬 빠른 최적화된 파이프라인.
    - 네이티브 해상도:기기의 네이티브 해상도를 완전 지원하여 가능한 최고의 시각적 선명함과 GPU 효율 제공.
