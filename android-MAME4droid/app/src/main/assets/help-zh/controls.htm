操作

这些是默认设置，如果您更改了手柄映射，则可能不总是适用。

虚拟手柄

  • 屏幕上的手柄和按钮
  • 用于模拟控制的触摸鼠标。第一根手指移动。第二根手指为左键。第三根手指为右键。
  • 光枪（点击屏幕；必须在 选项->设置->输入->触摸光枪 中启用）

游戏手柄

  • 手柄上的 A、B、C、D、E、F 按钮对应按钮 1 到 6
  • SELECT（投币）按钮投币/加币
  • START 按钮开始 1P 游戏

特殊

  • START+上 开始 2P 游戏。START+右 开始 3P 游戏。START+下 开始 4P 游戏
  • SELECT+上 投入 2P 币。SELECT+右 投入 3P 币。SELECT+下 投入 4P 币
  • START+SELECT 进入游戏的 MAME 菜单（视频选项、DIP 开关等）
  • A+START 是快进（按住时，以禁用节流和最大跳帧运行游戏）。
  • B+START 切换 UI 控制（对于有键盘输入的模拟系统，启用或禁用 UI 控制）。
  • R1+START 读取存档
  • L1+START 保存存档
  • START+A+B 是维修模式
  • SELECT+A+B 是软复位
  • L1 或 A+上 是游戏向上翻页
  • R1 或 A+下 是游戏向下翻页

注意：使用 SELECT(投币) 或 START 进行界面导航。
