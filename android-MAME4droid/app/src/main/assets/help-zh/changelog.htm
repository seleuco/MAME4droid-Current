MAME4droid 更新日志

开发：
  • 2026-07-08 版本 1.37 添加了点对点联机对战：
    - 两种模式：锁步（完美同步，输入延迟较高）和回滚（低延迟，适合快速反应的游戏）。
    - 自动回退：不符合存档大小/RAM 预算，或检测到设备间存档布局不匹配的回滚会话，会自动回退到锁步。
    - 游戏中重新同步：如果回滚会话明显偏离，可从联机对战对话框强制两台设备恢复完美同步。
    - 只有主机选择会话模式；客户端只需加入并自动采用。
    刷新并扩展了应用内帮助：新增联机对战和渲染器与着色器页面，以及涵盖 Android 10 起使用的 Storage Access Framework (SAF) 文件夹选择器的更新版安装指南。
  • 2026-24-4 版本 1.35 添加了具有显著改进的全新原生视频渲染器：
    - 矢量游戏：大幅提升的渲染质量，带来更清晰、更真实的外观。
    - 着色器：自定义着色器现在只应用于游戏画面，保持 MAME 界面/OSD 干净不受影响。
    - 性能：优化的管线，在使用高分辨率 artwork 和边框时快得多。
    - 原生分辨率：完全支持设备的原生分辨率，提供尽可能最佳的视觉清晰度和 GPU 效率。
