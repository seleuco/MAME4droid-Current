Controles

Estas são as configurações padrão e podem nem sempre se aplicar se você alterar os mapeamentos do controle.

Controle virtual

  • Controle e botões na tela
  • Mouse por toque para controle analógico. Primeiro dedo para mover. Segundo dedo é o botão esquerdo. Terceiro dedo é o botão direito.
  • Pistola de luz (toque na tela; deve ativar em Opções->Configurações->Entrada->Pistola de luz por toque)

Gamepad

  • Os botões A, B, C, D, E, F do controle correspondem aos botões 1 a 6
  • O botão SELECT (ficha) insere ficha/adiciona crédito
  • O botão START inicia o jogo de 1 jogador

Especiais

  • START+CIMA inicia o jogo de 2 jogadores. START+DIREITA de 3 jogadores. START+BAIXO de 4 jogadores
  • SELECT+CIMA insere créditos de 2P. SELECT+DIREITA de 3P. SELECT+BAIXO de 4P
  • START+SELECT acessa o menu do MAME do jogo (opções de vídeo, chaves DIP, etc.)
  • A+START é avanço rápido (enquanto pressionado, executa o jogo sem limitação de velocidade e com o máximo de salto de quadros).
  • B+START alterna os controles da interface (para sistemas emulados com entrada de teclado, ativa ou desativa o controle da interface).
  • R1+START carrega um estado salvo
  • L1+START salva um estado
  • START+A+B é modo de serviço
  • SELECT+A+B é reinício suave
  • L1 ou A+CIMA é AVANÇAR PÁGINA no jogo
  • R1 ou A+BAIXO é RECUAR PÁGINA no jogo

Observação: use SELECT(ficha) ou START para navegar pela interface.
