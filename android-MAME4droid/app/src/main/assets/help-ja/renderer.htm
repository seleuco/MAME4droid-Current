レンダラーとシェーダー

MAME4droid は 2 つの異なるレンダラーでゲームを描画できます。 オプション->設定->レンダラーとシェーダー->ネイティブレンダラー（シェーダーと HDR）で 切り替えます。

2 つのレンダラー

  • レガシー（ソフトウェア）レンダラー — デフォルト：古く、極めてよくテスト された描画パス。シェーダーや HDR のサポートはありませんが、低スペックや古いハードウェアを含む すべてのデバイスで最も安全で互換性の高い選択肢です。最小限の手間で動作させたいだけの場合、または デバイスがネイティブレンダラーに苦労する場合は、これを使い続けてください。
  • ネイティブレンダラー（シェーダーと HDR）：MAME 自身の現代的な描画パイプライン。 オンにするとシェーダー効果、ベクターゲームの True HDR、通常（ラスター）ゲームの Auto HDR、そして 全般的にベクターゲームのはるかに高品質な見た目が解放されます（下記参照）。レガシーレンダラーより 多くの GPU パワーを必要としますが、そこそこ現代的なデバイスなら、高解像度の artwork/ベゼルを使う 場合にはより高速でもあります。そのパイプラインはその用途に特別に最適化されているためです。

重要：ネイティブレンダラーを有効にしたら、 オプション->設定->エミュレーション->エミュレート解像度もネイティブに 設定してください。これにより高解像度の artwork/ベゼルがパフォーマンス低下なしにフル品質で描画されます — ネイティブレンダラー使用中に低い固定解像度のままにしておくと、せっかく解放した追加の画質を 無駄にします。

シェーダー

ネイティブレンダラーを有効にすると、レンダラーとシェーダー->エフェクトシェーダーで、 エミュレート画像の上に適用される後処理効果（スキャンライン/CRT 風フィルターなど）を選べます。利用可能な リストはデバイス/ビルドによります。無効にするにはなしを選択します。シェーダーは実際の ゲーム画面のみに影響します — MAME 自身のメニューとオンスクリーン表示は鮮明でフィルターがかからない ままなので、これを変更しても MAME の UI が読みにくくなることはありません。

ベクターゲーム：高度な CRT シミュレーション

ベクターゲーム（Asteroids、Tempest、Star Wars、Battlezone など、ピクセルグリッドではなく線で画像を 描く）はネイティブレンダラーから最も恩恵を受けます。ベクター高度エフェクトを有効にすると、 MAME4droid は単に平坦なアンチエイリアス線を描くのではなく、実際のベクター CRT モニターがそれらの線を どう描いていたかを物理ベースでシミュレートします：

  • 光学ブルーム：明るいベクター周辺の管ガラス内部での光散乱/ハロー。
  • 蛍光体の残光（トレイル）：電子ビームが残す、消えていく余韻。本物の蛍光スクリーンの ように。
  • 電子ビームダイナミクス：短いストローク（テキスト、小さな宇宙船のディテール）が 本物のハードウェアとまったく同じように、より明るく焼き付きます。
  • コーナーバーン：ビームが鋭いコーナーで減速する明るい「溶接点」。
  • 磁気ジッター/フリッカー：微妙なコイルノイズと電圧変動。完璧すぎるベクター画像 ではなく、本物の不完全な CRT の見た目のために。

見た目を微調整したい場合、これらのそれぞれにベクターエフェクトキャリブレーション（エキスパート モード）で独自のキャリブレーションスライダーがあります。迷ったらCRT デフォルトを復元 ですべてリセットできます。FBO 半解像度はベクターのグローパスを半解像度で描画して GPU/バッテリーを節約しますが、コア線がわずかに鮮明でなくなる代償があります — 非力なデバイスでは 良いトレードオフです。

HDR

デバイスとディスプレイが HDR に対応している場合、HDR 設定->HDR ディスプレイを有効化で パネルのピーク輝度と広色域が解放されます：

  • ベクターゲーム（True HDR）：蛍光体の明るさが標準ダイナミックレンジの白にクランプ されるのではなく、実際の HDR 輝度で描画され、元の CRT 蛍光体が実際にどれほど強烈に見えたかに近づき ます。CRT 基本/最大輝度が基準とピークの明るさ（ニト単位）を調整します。ベクター artworks を暗くするはゲーム周りのベゼルを暗くして、そのコントラストをさらに際立たせます。
  • ラスターゲーム（Auto HDR）：通常（非ベクター）のゲームとその artwork は標準 ダイナミックレンジのコンテンツなので、MAME4droid は逆トーンマッピングを使ってそれらを HDR 輝度へ オプションでアップグレードできます（Auto HDR を有効化（ラスター））。Auto HDR 強度 がハイライトの拡張の強さを制御し、ラスターペーパーホワイト基準が快適さのための基準の 明るさを設定します。

注意：HDR 品質はデバイス/パネルによって大きく異なります — 結果は主に行儀の 良い実装（例：Google Pixel）を中心に調整されており、他のハードウェアでは、良くも悪くも異なって見える ことがあります。キャリブレーションが狂ったときのためにHDR デフォルトを復元があります。
