操作

これらはデフォルト設定であり、コントローラーのマッピングを変更した場合は常に適用されるとは限りません。

バーチャルパッド

  • 画面上のコントローラーとボタン
  • アナログ操作用のタッチマウス。1 本目の指で移動。2 本目の指が左ボタン。3 本目の指が右ボタン。
  • ライトガン（画面をタップ。オプション->設定->入力->タッチライトガン で有効にする必要があります）

ゲームパッド

  • コントローラーの A、B、C、D、E、F ボタンはボタン 1～6 に対応
  • SELECT（コイン）ボタンでコイン投入/クレジット追加
  • START ボタンで 1P ゲーム開始

特殊

  • START+上 で 2P ゲーム開始。START+右 で 3P ゲーム開始。START+下 で 4P ゲーム開始
  • SELECT+上 で 2P クレジット投入。SELECT+右 で 3P クレジット投入。SELECT+下 で 4P クレジット投入
  • START+SELECT でゲームの MAME メニュー（映像オプション、DIP スイッチなど）にアクセス
  • A+START は早送り（押している間、スロットリング無効と最大フレームスキップでゲームを実行）。
  • B+START は UI コントロールの切り替え（キーボード入力のあるエミュレートシステムで、UI コントロールを有効/無効化）。
  • R1+START でステートをロード
  • L1+START でステートをセーブ
  • START+A+B はサービスモード
  • SELECT+A+B はソフトリセット
  • L1 または A+上 はゲームのページアップ
  • R1 または A+下 はゲームのページダウン

注意：UI ナビゲーションには SELECT(コイン) または START を使用してください。
