MAME4droid 変更履歴

開発：
  • 2026-07-08 バージョン 1.37 ピアツーピアのネットプレイを追加：
    - 2 つのモード：ロックステップ（完璧に同期、入力遅延が大きい）とロールバック（低遅延、素早い反応が必要なゲームに最適）。
    - 自動フォールバック：ステートセーブのサイズ/RAM 予算に収まらない、またはデバイス間でステートのレイアウトの不一致を検出したロールバックセッションは、自動的にロックステップにフォールバックします。
    - ゲーム中の再同期：ロールバックセッションが目に見えてずれた場合、ネットプレイダイアログから両方のデバイスを完璧な同期に強制的に戻します。
    - ホストのみがセッションモードを選びます。クライアントは参加して自動的にそれを採用するだけです。
    アプリ内ヘルプを刷新・拡充：新しいネットプレイとレンダラーとシェーダーのページ、および Android 10 以降で使われる Storage Access Framework (SAF) フォルダピッカーを扱う更新されたインストールガイド。
  • 2026-24-4 バージョン 1.35 大幅な改善を伴う新しいネイティブビデオレンダラーを追加：
    - ベクターゲーム：より鮮明で本物らしい見た目のために描画品質を大幅に向上。
    - シェーダー：カスタムシェーダーはゲーム画面のみに適用されるようになり、MAME の UI/OSD をきれいなまま保ちます。
    - パフォーマンス：高解像度の artwork やベゼルを使うときにはるかに高速な最適化パイプライン。
    - ネイティブ解像度：デバイスのネイティブ解像度を完全サポートし、可能な限り最高の視覚的な明瞭さと GPU 効率を提供。
